1. 系統環境：Windows10 64-bit
   編譯器環境：CodeBlocks + MinGW

2. 請先打開CodeBlocks，點選上方頁籤的 File -> Open...，選擇 hw2 資料夾下的 hw2.cbp 檔案，這時候會開啟整個專案。

3. 接著，直接點選 Build and Run 就能執行程式。

4. 執行 hw2.exe 程式後，請在命令提示字元視窗的標題列點右鍵->內容，進入"版面配置"頁籤，將螢幕緩衝區和視窗大小的寬度改為180，然後按下確定；再來到"字型"頁籤，就能把畫面文字大小調大，大小請勿超過24。

5. 最後，請將命令提示字元視窗"最大化"(視窗右上角的中間按鈕)，才不會有換行的問題。

※注意：執行檔位於 hw2/bin/debug/hw2.exe，如果要直接點開執行，務必將 exe 檔移動至 hw2 資料夾下，否則會抓不到資料庫 test.db。
